WHAT WE PROVIDE

ImmunoIntegrate (go/immunointegrate) is a multi-omics integration tool that leverages evidence from multiple data sources. A query defining biological conditions of interest (e.g. diseases, organisms, treatments, tissues or cell types) is used to run an integration analysis on omics data and internally generated disease-related gene signatures. The output consists of several metrics meant to guide the user to potentially interesting targets. Selected targets can be imported into ImmunoCompare for further interrogation and subsequent target validation. Currently, ImmunoIntegrate contains 20 microarray datasets, 31 bulk RNASeq datasets, 15 single-cell RNASeq datasets, 22 proteomics datasets, and 3 AbbVie target lists spanning 12 diseases.
